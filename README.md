# Rep Daddy v4 Premium

GitHub Pages ready premium version.

## Files
- index.html
- manifest.webmanifest
- sw.js
- icon-180.png
- icon-192.png
- icon-512.png

## Deploy
1. Upload all files to repo root.
2. Settings → Pages
3. Deploy from branch → main → /(root)
4. Open the GitHub Pages URL in Safari.
5. Share → Add to Home Screen
