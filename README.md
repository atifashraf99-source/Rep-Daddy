## Upload to GitHub Pages
1. Create a new GitHub repo, for example `rep-daddy`.
2. Upload all files from this folder to the repo root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save and wait for the Pages URL.

## Install on iPhone
1. Open the GitHub Pages URL in Safari.
2. Tap **Share**.
3. Tap **Add to Home Screen**.
4. Open the new icon from the Home Screen.

## Notes
- The app stores data locally on each user's device.
- Backup/restore is for moving or protecting data.
- Each friend gets their own separate local data on their own device/browser.
